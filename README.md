# Описание
Плагин - Indentation helper

Предназначение - помощь в форматировании отступов в выделенном тексте, преобразует горизонтально перечисляемые в вертикально перечисляемые

Описание работы - необходимо выделить нужные элементы и нажать комбинацию клавиш `ctrl+shift+r`, элементы должны перечисляться через `,`. После чего горизонтально перечисляемые элементы станут вертикально перечисляемыми

# Описание
Плагин - Indentation helper

Предназначение - помощь в выравнивании перечисляемых элементов во время написание кода

Описание работы - необходимо выделить нужные элементы, после чего нажать комбинацию клавиш `ctrl+shift+r`, элементы должны перечисляться через `,`

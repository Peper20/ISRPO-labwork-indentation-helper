# Техническое задание: расширение для VS Code - Indentation Helper

## 1. Общие сведения

### 1.1. Наименование
_расширение для VS Code:_ Indentation Helper

### 1.2. Назначение
Расширение предназначено для автоматического форматирования выделенного текста, преобразуя горизонтально перечисляемые элементы через запятую в вертикально перечисляемые.

## 2. Функциональные требования

- Преобразование горизонтальных списков (элементы через запятую) в вертикальные списки
- Сохранение исходного отступа строки при форматировании
- Работа только с выделенным фрагментом текста

## 3. Технические требования

- IDE - VS Code версии 1.60.0+
- Язык разработки - JavaScript
